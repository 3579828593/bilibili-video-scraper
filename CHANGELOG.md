# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02

### 🎉 首次发布
- ✅ **WBI 签名模块** — 完整实现 B 端 WBI 签名算法（混淆表 + MD5）
- ✅ **视频抓取模块** — 支持自动分页、智能重试、风控检测
- ✅ **数据分析模块** — 60+ 游戏别名库、9 大类型分类、Markdown 报告生成
- ✅ **CLI 入口** — 完整的命令行参数支持（argparse）
- ✅ **单元测试** — 覆盖签名算法、游戏提取、类型分类、数据解析
- ✅ **完整文档** — README、示例说明、API 文档
- ✅ **MIT 开源协议**

### 核心特性
- 无需 Cookie / 登录即可抓取公开数据
- 自动处理分页（支持数百甚至数千条视频）
- 失败页面记录与补抓机制
- 一键生成结构化分析报告

### 已知限制
- 受 B 站 API 频率限制，默认 0.5s 间隔
- 游戏识别依赖内置别名库，无法覆盖所有游戏
- 分析报告为纯文本 Markdown（无图表可视化）
