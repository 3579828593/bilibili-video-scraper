# 安全策略

## 报告安全漏洞

如果你发现了安全漏洞，请**不要**通过公开 Issue 报告。

请发送邮件至 3579828593@qq.com，邮件标题以 `[SECURITY]` 开头，描述：

1. 漏洞类型和严重程度
2. 复现步骤
3. 影响范围
4. 建议的修复方案（如有）

## 响应时间

- 确认收到：48 小时内
- 初步评估：7 天内
- 修复发布：根据严重程度，30 天内

## 安全使用建议

1. 不要在代码中硬编码任何 Cookie 或登录凭证
2. 使用合理的请求间隔（默认 0.5 秒）
3. 不要将抓取数据用于商业用途
4. 遵守 Bilibili 的服务条款
